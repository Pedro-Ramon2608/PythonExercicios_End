🐍 Exercícios de Python (Curso em Vídeo)

Este repositório é o registro completo da minha jornada no curso de Python 3, ministrado pelo professor Gustavo Guanabara no portal Curso em Vídeo. O projeto contém 115 desafios resolvidos, cobrindo desde os conceitos mais básicos até estruturas de dados avançadas e modularização.

#
🌎 Estrutura dos Mundos

O curso é dividido em três grandes etapas (Mundos), cada uma focada em um pilar da linguagem:

🔹 Mundo 1: Fundamentos (Desafios 001 ao 035)

Onde tudo começou. Foco em aprender a sintaxe e a lógica básica.

Tratamento de dados e aritmética.

Uso de módulos (Bibliotecas).

Manipulação de Strings.

Estruturas condicionais simples e compostas (if/else).

🔹 Mundo 2: Estruturas de Controle (Desafios 036 ao 071)

Foco em controle de fluxo e repetições.

Condicionais aninhadas (elif).

Estruturas de repetição: for e while.

Interrupção de laços (break).

🔹 Mundo 3: Estruturas Compostas e Módulos (Desafios 072 ao 115)

A fase mais avançada, lidando com grandes volumes de dados e organização profissional.

Tuplas, Listas e Dicionários: Armazenamento e manipulação de dados complexos.

Funções: Reaproveitamento de código e escopo.

Modularização e Pacotes: Organização do projeto em múltiplos arquivos.

Tratamento de Erros e Exceções: Criação de sistemas resilientes.

* Sem dúvidas manipular strings foi razoalvemente um dos temas complexo no inicio e as listas com toda certeza. Além disso, os top 3 exercícios mais desafiadores que enfentei foram:
* Sequência de Fibonacci (63)
* Simulador de caixa eletrônico (exercício 71)
* Cadastro de Pessoas e Análise (exercicio 115)
